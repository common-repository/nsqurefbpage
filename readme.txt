== Description ==
You can add facebook page any where in your post as well as in theme source code.

Add shortcode for display facebook page on your website.
Use [fbpage] shortcode for display facebook page plugin in post.
Use <?php echo do_shortcode('[fbpage]'); ?> code for add facebook page plugin in theme.

Form description 

Facebook Page URL : Here we add facebook page URL.
Width : Here we add width of plugin. if we want.
Height: Here we add height of plugin. if we want.
Use Small Header : set true. it Uses a smaller version of the page header.
Adapt to plugin container width : set true. Plugin will try to fit inside the container.
Hide Cover Photo : set true. if you want to hide the cover photo in the header.
Show Friend's Faces : set true. if you want to show profile photos when friends like this.
Show Page Posts : set true. if you want to Show posts from the Page's timeline.

Major features
1. Show who likes your Page with real people's profile images rather than just a number. People visting your Page will see a count of friends that like the Page as well as their profile photos.
2. You can show posts from the Page's timeline in your Page plugin implementation. This is a great way to keep people up to date with what is going on on your Page.
3. The Page plugin works with responsive, fluid and static layouts. You can use media queries or other methods to set the width of the parent element.
4. You can like and share your facebook page.
